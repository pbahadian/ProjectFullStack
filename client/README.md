# Final Project - backend + frontend 

Proyecto de e-commerce con base de datos integrada (express) desde MySQL.

<img src="/src/assets/Screenshot 2026-07-20 at 19.26.56.png>

El objetivo es simular un e-commerce real, donde se pueda probar el proceso de compra y comprobar que la navegación funciona entre los distintos views y stores.

El visual de la página busca priorizar el producto, por lo que hemos utilizado Grid con un estilo de portfolio con cards y colores y formas limpias y poniendo campos de informacion claros para faciltar la navegación.

La base de datos llama imágenes desde la url https://paulabahadian.wordpress.com/


Hemos utilizado Pinia para crear una  (store) con objetos favoritos. 

Tenemos una parte de ADMIN , con un FORM en el que puedes añadir piezas nuevas a la página.

La página del Carrito tiene un Checkout de pago simulado.
